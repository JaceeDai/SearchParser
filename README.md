# SearchParser

## 版本

Gradle
```gradle
    implementation 'io.github.jaceed:searchparser:1.0.0'
```
或

Maven
```gradle
    <dependency>
       <groupId>io.github.jaceed</groupId>
       <artifactId>searchparser</artifactId>
       <version>1.0.0</version>
    </dependency>
```

## 说明

这是一个字符串匹配（搜索）的工具类。
